# mRIXS

This submodule contains the main mRIXS processing and analysis utilities.

## Main Files

- `mrixs_pipeline.py`
  Main pipeline script for mRIXS data processing.
- `rixs_phonon_analyzer.py`
  Base analyzer for phonon-related mRIXS interpretation.
- `rixs_phonon_analyzer_v2.py`
  Revised analyzer version.
- `rixs_phonon_analyzer_v3.py`
  Further refined analyzer version.
- `rixs_phonon_analyzer_v3_1.py`
  Latest incremental analyzer version currently stored in this directory.
- `figure_generation.ipynb`
  Notebook for producing figures or inspection plots from processed outputs.

## Notes

- The numbered analyzer variants likely represent iteration history.
- If one version becomes preferred, the others can later be moved into an `archive/` or `legacy/` folder.
