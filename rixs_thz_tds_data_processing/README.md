# RIXS and THz-TDS Data Processing

This module organizes spectroscopy-oriented data-processing workflows.

## Submodules

- `mrixs/`
  mRIXS analysis scripts, pipelines, and figure-generation notebooks.
- `thz_tds/`
  THz-TDS processing notebook(s).

## Typical Use

- Preprocess spectroscopy datasets
- Extract spectral features or phonon-related signatures
- Generate publication-style plots and intermediate analysis outputs
