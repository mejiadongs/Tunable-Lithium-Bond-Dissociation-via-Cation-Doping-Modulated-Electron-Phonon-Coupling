# Phonon

This submodule organizes phonon-related calculations into two parts:

- `phonon/`
  Core phonon calculation and visualization utilities.
- `phonon_scattering/`
  Notebooks focused on phonon lifetimes, scattering behavior, and temperature analysis.

## Suggested Workflow

1. Build or load phonon calculation outputs in `phonon/`.
2. Export band, q-point, or element-resolved information.
3. Continue with lifetime and scattering analysis in `phonon_scattering/`.
