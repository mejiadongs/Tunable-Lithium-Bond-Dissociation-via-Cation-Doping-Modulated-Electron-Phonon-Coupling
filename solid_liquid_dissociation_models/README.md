# Solid and Liquid Dissociation Models

This module groups modeling workflows for bond dissociation in both solid-phase and liquid-phase environments.

## Submodules

- `solid/`
  Solid-state model notebooks.
- `liquid/`
  Adsorption, binding-energy, Gibbs free energy, and slab-freezing workflows for liquid-phase or interface studies.

## Suggested Workflow

1. Build or inspect candidate structures.
2. Run adsorption or dissociation modeling for the target phase.
3. Postprocess energies and free-energy trends.
4. Compare phase-dependent dissociation behavior across systems.
