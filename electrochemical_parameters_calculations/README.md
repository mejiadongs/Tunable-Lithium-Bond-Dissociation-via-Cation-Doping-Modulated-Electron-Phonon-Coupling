# Electrochemical Parameters Calculations

This module groups electrochemical-property analysis scripts and notebooks.

## Main Files

- `lithium_ion_transference_number_calculator.py`
  Script for estimating the lithium-ion transference number.
- `lithium_ion_conductivity_curve_fitting.ipynb`
  Notebook for fitting conductivity data and visualizing electrochemical trends.

## Typical Use

- Fit conductivity-related experimental data
- Estimate transport descriptors
- Export processed values for comparison with structural or spectroscopic results
