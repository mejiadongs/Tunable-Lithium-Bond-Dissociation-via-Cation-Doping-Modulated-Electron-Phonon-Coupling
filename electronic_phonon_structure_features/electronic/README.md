# Electronic

This submodule contains notebooks and scripts for electronic-structure feature extraction and surface-model preparation.

## Main Files

- `d_band_center.py`
  Calculates or assists with d-band-center analysis.
- `p_band_center.py`
  Calculates or assists with p-band-center analysis.
- `surface_li_modeling.ipynb`
  Notebook for Li-surface model construction or inspection.
- `surface_s_modeling.ipynb`
  Notebook for S-surface model construction or inspection.
- `data_extraction_and_visualization_preview.ipynb`
  Notebook for extracting calculated data and previewing plots.
- `d_band_calculation_backup.ipynb`
  Archived or alternative notebook related to d-band analysis.
- `p_band_calculation_backup.ipynb`
  Archived or alternative notebook related to p-band analysis.

## Notes

- The `_backup` notebooks appear to preserve alternative analysis paths or earlier working versions.
- If one notebook becomes canonical, it can later be promoted and the backup copy archived elsewhere.
