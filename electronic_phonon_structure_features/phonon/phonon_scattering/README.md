# Phonon Scattering

This directory contains notebooks for analyzing phonon lifetimes, scattering processes, and temperature-dependent behavior.

## Main Files

- `phonon_lifetime.ipynb`
  Main lifetime-analysis notebook.
- `phonon_scattering.ipynb`
  Main scattering-analysis notebook.
- `temperature_analysis.ipynb`
  Temperature-focused analysis notebook.
- `phonon_lifetime_backup.ipynb`
  Backup or alternate version of the lifetime workflow.
- `phonon_scattering_backup.ipynb`
  Backup or alternate version of the scattering workflow.
- `phonon_scattering_backup_2.ipynb`
  Second backup or alternate version of the scattering workflow.

## Notes

- The backup notebooks are kept for reproducibility and comparison.
- If desired later, historical notebooks can be moved into a dedicated `archive/` directory.
