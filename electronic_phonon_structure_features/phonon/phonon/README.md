# Core Phonon Workflows

This directory stores scripts and notebooks used for core phonon calculations, data extraction, and visualization.

## Main Files

- `phonopy_force_calc.py`
  Helper for force calculations used in phonon workflows.
- `plot_band.py`
  Band-structure plotting utility.
- `plot_qpoint_contrib.py`
  Q-point contribution visualization utility.
- `plot_qpoint_contrib_cluster_irrep.py`
  Extended q-point contribution analysis with cluster or irrep grouping.
- `export_element_bands_flexible.py`
  Exports element-resolved or customized band information.
- `epc_viz.py`
  Visualization helper for electron-phonon-coupling related outputs.
- `atom_index.py`
  Atom-index utility for structure-based mapping.
- `band.conf`
  Configuration file used in phonon-band workflows.
- `phonon_run_workflow.ipynb`
  Notebook for running or documenting the end-to-end phonon workflow.
